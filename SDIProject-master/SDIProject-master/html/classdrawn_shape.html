void drawMe ()
 drawnShape::drawMe Draws the selected shape. The area dragged accross to put the shape in is used to determine the points of each shape. For a rectangle, this is simply the start and end points. For a triangle, the middle of the shape start is used. For a trapezium, it is slightly indented for a quarter of the length. These points are added to the list which plots these points.
