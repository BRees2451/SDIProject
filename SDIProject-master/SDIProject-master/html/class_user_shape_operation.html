Detailed Description

The UserShapeOperation class handles all of the processes involved when using the shapes. This may include drawing and displaying the names.

void UserShapeOperation::Delete ( QGraphicsScene *  scene)

Delete is called when dealing with deleting a shape.

UserShapeOperation::Delete is repsonsible for removing shapes from the canvas. It will look through the shape list and see if there is one selected to then once found delete the shape.

handleMouseEvent will take the mouse event and decide whether to create a new instance of the shape that we are trying to draw or carry on drawing a shape that has been started.

UserShapeOperation::handleMouseEvent will take the mouse event at the time and then decide what to do with it. It can either create another object or continue drawing the same object depending on whether there is a shape still being drawn.
